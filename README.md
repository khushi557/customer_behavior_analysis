📊 Customer Behavior Dashboard (Power BI)

📌 Overview

An interactive Power BI dashboard that analyzes customer purchasing behavior, revenue trends, and subscription insights. The project helps uncover patterns and supports data-driven decision-making.

🎯 Key Objectives
Understand customer purchase behavior
Analyze revenue and average spending
Evaluate subscription-based trends
Derive actionable business insights

🛠️ Tech Stack
Power BI – Data visualization & dashboard creation
SQL – Data extraction and preprocessing
📈 Dashboard Features

✨ Interactive filters and slicers
📊 KPIs: Total Revenue, Average Spend, Customer Count
📉 Subscription status analysis
📌 Clear and intuitive visualizations

📂 Dataset

Customer dataset containing:
Purchase Amount
Subscription Status
Customer ID & Transaction Details

🚀 Getting Started
Clone this repository
Open the .pbix file in Power BI Desktop
Interact with the dashboard using filters

📊 Key Insights
Identified high-value customer segments
Compared revenue across subscription types
Observed trends in customer spending behavior
